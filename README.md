# **Simple File Handling with Python**

## **Project Description**
This Python-based project provides a simple file handling system using `.txt` files to store and manage a to-do list. It allows users to efficiently add, delete, update, and display items from a stored list. The project follows an object-oriented approach and ensures data persistence across sessions.

## **Features**
✅ Load and save data using `.txt` files  
✅ Add new items to the list  
✅ Delete existing items  
✅ Update items in the list  
✅ Display the stored list in a user-friendly format  
✅ Error handling for missing files and invalid input
